# LabFlow — Short Presentation Script

## 1. Problem

Schools and educational labs often track equipment with spreadsheets or informal messaging. This makes borrowing, maintenance, and fault tracking difficult.

## 2. Solution

LabFlow is a local-first laboratory equipment management platform built for real educational workflows. It combines inventory, QR codes, borrowing, maintenance, reservations, analytics, and a companion mobile app.

## 3. Distinctive value

- multi-lab support
- educational roles
- QR workflows
- maintenance and defect lifecycle
- local API + Android companion
- polished analytics dashboard

## 4. Live demo path

1. login
2. open `Presentation Demo Lab`
3. dashboard overview
4. inventory and containers
5. borrowing workflow
6. fault report workflow
7. reservations and calendar
8. leaderboard / student activity
9. AI helper

## 5. Why it matters

The app reduces operational friction in schools and gives teachers and technicians a shared source of truth for laboratory resources.

## 6. Closing line

LabFlow is not just a CRUD inventory app; it is an integrated laboratory operations system tailored to education.
